# quotation-automation

This package is part of the BTSA Bun workspace monorepo.

To install dependencies, run this from the repository root:

```bash
bun install
```

To run the backend from the repository root:

```bash
bun run dev:backend
```

To run backend-only commands from this package directory:

```bash
bun run dev
bun run typecheck
```

This project was created using `bun init` in bun v1.3.9. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
